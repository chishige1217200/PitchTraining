# PitchTraining

## How to setup
1. ライブラリをインストールする
```sh
pip install -r requirements.txt
```

2. FluidSynthをインストールする

[公式サイト](https://github.com/FluidSynth/fluidsynth/wiki/Download)で案内されている、Chocolateyを介した導入が簡単です。自動でパスも通してくれます。

3. sf2ファイル（音源）を.envファイルで指定する。

.env.sampleを.envにリネームして、ファイル内でsf2ファイルへのパスを指定してください。
